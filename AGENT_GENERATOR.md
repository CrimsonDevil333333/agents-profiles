## What To Do Instead

### For Users (Every Session)

Give your AI this file at the start of every chat:

> **[`INIT.md`](./INIT.md)**

This file establishes the multi-agent system, defines the Orchestrator role, and includes a full delegation matrix mapping tasks to agents.

### For AI Agents (Project Analysis)

Load this skill to analyze a project and select the right agents:

> **[`skill.md`](./skill.md)**

The skill analyzes your project's tech stack, architecture, and domain, then maps it to the closest-matching agents from the 118 existing profiles. It recommends a roster for your confirmation. It only generates a new profile when no existing one fits.
